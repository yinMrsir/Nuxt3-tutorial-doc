# 介绍

现在Vue3已经非常稳定了，而且用在生产环境的项目也很多，Nuxt2升级到Nuxt3后也很好的支持了Vue3, 同时Nuxt3官网文档也是非常齐全的，但是在开发过程中仍会遇到很多其他的问题，如其他组件模块安装使用等。所以编写了此文档，以项目实战为主，从0开始学习Nuxt3，边做边学，实实在在的完成一个项目并上线。
